# Firecracker SDK

> [!WARNING]
> 
> This SDK is built on firecracker **v1.14.1**.
> Compatibility with other versions is not guaranteed.
> 
> The test suite is currently incomplete and not recommended for use in production environments.
> 
> This SDK is under active development.

## Prerequisites

- [Firecracker](https://github.com/firecracker-microvm/firecracker)
- [Linux KVM](https://linux-kvm.org/page/Main_Page)
- Kernel image (recommended: Linux v6.1.y)
- Rootfs
